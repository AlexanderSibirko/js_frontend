Встроенные директивы, что такое методы и как с ними работать, работа с аднными, добавление стилизации.

Директивы - специальный атрибут html или компонента с приставкой v-, который привязывает изменение этого элемента к какому-то js выражению.

-- v-on
v-on:click
v-on:input
v-on:focus
v-on:mouseover
v-on:keyup

-- v-bind
Данная директива - позволяет "привязать" значение атрибута. Простыми словами передать значение из data в код
:атрибут="переменная", :style="объектСтилей" - так проще

-- v-model
Эта директива помогает связать элемент для ввода с какой-то переменной в ваших данных. Причём сдеалть это в двух направлениях: изменени воода будет менять data и наоборот.
v-model.trim - убирает лишние пробелы
v-model.lazy - синхронизирует ввод с данными по событию input, чтобы использовать для синхронизации после события change
v-model.number - для автоматического приведения введённого значения к Number
